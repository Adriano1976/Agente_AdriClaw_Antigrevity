# Artigos e Newsletters no LinkedIn

## Quando usar artigo vs. newsletter vs. post longo

| Formato | Comprimento | Objetivo | SEO |
|---|---|---|---|
| Post de texto | até ~3.000 chars | Engajamento rápido | Não |
| Artigo | 500–2.000 palavras | Autoridade, aprofundamento | Sim |
| Newsletter | 300–1.500 palavras | Assinantes fiéis, recorrência | Parcial |

---

## Artigos

Artigos no LinkedIn são indexados pelo Google. Use para:
- Conteúdo evergreen (sempre relevante)
- Posicionamento como especialista
- Temas que precisam de profundidade

**Estrutura recomendada:**
1. Título (H1) — use palavra-chave principal
2. Lead (1–2 parágrafos) — por que esse tema importa
3. Corpo com subtítulos (H2/H3)
4. Conclusão com CTA
5. Bio curta no final (quem você é, o que faz)

---

## Newsletter

A Newsletter do LinkedIn envia notificação para seus seguidores. É um dos poucos canais com alcance garantido.

**Dicas:**
- Cadência consistente (semanal ou quinzenal)
- Nome forte para a newsletter (ex: "O Briefing de Liderança", "Tech para Humanos")
- Cada edição deve ter valor independente — o leitor não precisa ter lido as anteriores
- Sempre terminar com "próxima edição: [tema]" para gerar antecipação

**Estrutura simples que funciona:**
```
[Título da Edição]

→ Destaque da semana (1 insight principal)
→ O que estou lendo/testando
→ Uma dica rápida aplicável hoje
→ Pergunta para a comunidade

[Assine / Compartilhe]
```
